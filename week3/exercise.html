Latihan Minggu 3: Input dan Percabangan

1. Mengecek Lulus KKM

Buatlah program yang bisa memberikan info apakah nilai yang diberikan lulus KKM atau tidak.
Batas nilai adalah 0-100.
Batas nilai lulus adalah harus lebih besar atau sama dengan 80.

  • Ketika dijalankan, dia akan meminta inputan nilai.
  • Begitu nilai diinput, program harus meng-output sebuah kata penanda lulus atau tidak.
  • Kata penanda lulus hanya ada dua: "Lulus" atau "Tidak Lulus", harus persis, tidak boleh kata yang lain. Huruf besar dan kecil juga harus sama.
  • Output "Lulus" jika nilai tersebut lulus KKM, atau output "Tidak Lulus" jika nilai tersebut tidak lulus KKM.
  • Anggap bahwa semua nilai yang dimasukkan pasti ada di rentang 0 sampai 100 (tidak perlu dicek lagi).
Contoh #1:
Input (Terminal) 
90
Output (Terminal) 
Lulus
Contoh #2:
Input (Terminal) 
74
Output (Terminal) 
Tidak Lulus

2. Konversi Nilai Angka ke Nilai Huruf

Buatlah program yang bisa memberikan nilai huruf dari nilai angka yang diberikan.
Batas nilai adalah 0-100.
Berikut adalah aturan mengkonversi nilai:

  • ≥ 90 = A
  • 80-89 = B
  • 70-79 = C
  • 60-69 = D
  • 50-59 = E
  • < 50 = F

Jika nilai lebih dari 100 atau kurang dari 0, keluarkan "Nilai Tidak Valid!" (persis).

  • Ketika dijalankan, dia akan meminta inputan nilai.
  • Begitu nilai diinput, program harus meng-output sebuah huruf atau sebuah kata.
  • Gunakan aturan di atas untuk menentukan output.
  • Anggap bahwa semua nilai yang dimasukkan pasti angka (tidak mungkin string).
Contoh #1:
Input (Terminal) 
93
Output (Terminal) 
A
Contoh #2:
Input (Terminal) 
63
Output (Terminal) 
D
Contoh #3:
Input (Terminal) 
102
Output (Terminal) 
Nilai Tidak Valid!
Contoh #4:
Input (Terminal) 
-17
Output (Terminal) 
Nilai Tidak Valid!

3. Mengecek Nama "Budi"

Buatlah program yang bisa mengecek apakah nama yang diinput adalah "Budi" atau bukan.

  • Ketika dijalankan, program akan meminta input nama.
  • Setelah nama diinput, program harus meng-output sebuah kata penanda.
  • Jika nama yang diinput adalah "Budi" (persis, huruf besar dan kecil harus sama), outputkan "Nama ditemukan".
  • Jika nama yang diinput bukan "Budi", outputkan "Nama tidak ditemukan".
  • Output harus persis seperti di atas, tidak boleh ada tambahan kata atau huruf.
Contoh #1:
Input (Terminal) 
Budi
Output (Terminal) 
Nama ditemukan
Contoh #2:
Input (Terminal) 
budi
Output (Terminal) 
Nama tidak ditemukan
Contoh #3:
Input (Terminal) 
Andi
Output (Terminal) 
Nama tidak ditemukan
