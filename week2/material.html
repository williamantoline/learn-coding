Materi Minggu 2: Python Dasar untuk Pemula

1. Komentar di Python

Komentar adalah bagian dari kode yang tidak dijalankan oleh komputer. Komentar digunakan untuk menjelaskan maksud kode, memberi catatan, atau menonaktifkan kode sementara. Komentar sangat penting agar kode mudah dipahami oleh diri sendiri maupun orang lain.

Komentar Satu Baris

Untuk membuat komentar satu baris, gunakan tanda # di awal baris. Semua teks setelah # akan diabaikan oleh Python.

# Ini adalah komentar satu baris
print("Hello") # Ini juga komentar di akhir baris

Komentar Multi-Baris

Untuk komentar lebih dari satu baris, bisa menggunakan beberapa # di setiap baris, atau menggunakan triple quotes """ ... """.

# Ini komentar baris pertama
# Ini komentar baris kedua
# Ini komentar baris ketiga
"""
Ini komentar multi-baris.
Biasanya digunakan untuk dokumentasi atau penjelasan panjang.
"""
Tips: Selalu beri komentar pada bagian kode yang penting atau rumit agar mudah dipahami!

2. Penamaan Variabel

Variabel adalah "wadah" untuk menyimpan data. Nama variabel harus mengikuti aturan agar tidak error dan mudah dibaca.

  • Nama variabel harus diawali huruf atau garis bawah (_).
  • Tidak boleh ada spasi. Gunakan _ sebagai pemisah kata.
  • Nama variabel tidak boleh menggunakan karakter khusus seperti !, @, #, dll.
  • Nama variabel sebaiknya jelas dan mudah dimengerti.
  • Python membedakan huruf besar dan kecil (nama dan Nama adalah variabel berbeda).
nama_lengkap = "Andi"
tinggi_badan = 170
usia = 25

Contoh Nama Variabel yang Salah

2nama = "Budi" # Salah, tidak boleh diawali angka
tinggi badan = 170 # Salah, tidak boleh ada spasi
nama-lengkap = "Cici" # Salah, tidak boleh ada tanda -
Tips: Gunakan nama variabel yang singkat tapi jelas, misal umur, nilai_ujian, is_active.

3. Menggunakan Variabel dalam print() dan Penggabungan String

Fungsi print() digunakan untuk menampilkan data ke layar. Kita bisa menggabungkan variabel dengan string menggunakan tanda + (harus sama tipe datanya) atau menggunakan koma , di dalam print().

Menggabungkan dengan +

nama = "Andi"
usia = 25
print("Nama saya " + nama + ", usia saya " + str(usia))

Catatan: Jika ingin menggabungkan string dengan angka, angka harus diubah dulu ke string dengan str().

Menggabungkan dengan Koma

print("Nama saya", nama, "usia saya", usia)

Dengan koma, Python otomatis memisahkan dengan spasi dan tidak perlu mengubah tipe data.

Contoh Lain

tinggi = 170
print("Tinggi badan saya adalah", tinggi, "cm")
Tips: Jika ingin format yang lebih rapi, bisa gunakan f-string:
print(f"Nama saya {nama}, usia saya {usia}")

4. Tipe Data Dasar

Tipe data adalah jenis data yang bisa disimpan di variabel. Python punya beberapa tipe data dasar:

  • Integer: Bilangan bulat, misal 5, -20, 1000
  • Float: Bilangan desimal, misal 2.7, -20.01, 3.14
  • String: Teks, misal "hello", "123", "Python itu mudah"
  • Boolean: Nilai logika, hanya True atau False

Contoh Penggunaan Tipe Data

a = 5 # integer
b = 2.7 # float
c = "hello" # string
d = True # boolean

Tips Mengenali Tipe Data

  • Angka bulat tanpa titik adalah integer.
  • Angka dengan titik adalah float.
  • Teks diapit tanda kutip ("" atau '') adalah string.
  • True dan False adalah boolean.
Tips: Gunakan fungsi type() untuk mengecek tipe data:
print(type(a))

5. Operasi Boolean Dasar

Boolean adalah tipe data yang hanya punya dua nilai: True (benar) dan False (salah). Boolean sering digunakan untuk membuat keputusan dalam program.

Operator Boolean

  • AND (and): Hasilnya True hanya jika kedua nilai True.
  • OR (or): Hasilnya True jika salah satu nilai True.
  • NOT (not): Membalik nilai boolean. not True jadi False, not False jadi True.

Contoh Operasi Boolean

print(True and True) # True
print(True and False) # False
print(False and True) # False
print(False and False) # False
print(True or True) # True
print(True or False) # True
print(False or True) # True
print(False or False) # False
print(not True) # False
print(not False) # True

Contoh Penggunaan dalam Program

umur = 19
print(umur >= 18) # True, karena 18 lebih besar dari 17
print(umur < 18) # False
cuaca = "cerah"
hari_ini = "minggu"
print(cuaca == "cerah" and hari_ini == "minggu") # True
Tips: Boolean sangat penting untuk membuat keputusan, misal dalam if dan else.
