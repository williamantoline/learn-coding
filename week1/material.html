1. Apa Itu Programming?

Pemrograman adalah proses memberikan instruksi yang sangat, sangat, sangat detail kepada komputer.

Bayangkan Anda ingin membuat kue. Anda tidak hanya mengatakan "buat kue yang enak," kan? Anda akan memberikan resep yang berisi setiap langkah dengan jelas: "ambil 200 gram tepung," "campurkan dengan 100 gram gula," "aduk selama 5 menit," dan seterusnya.

Komputer sama seperti itu. Komputer tidak punya "akal sehat" untuk menebak maksud kita. Ia hanya bisa mengikuti instruksi. Jadi, tugas kita sebagai programmer adalah menuliskan "resep" (yang kita sebut kode) yang tidak ambigu, langkah demi langkah, agar komputer bisa melaksanakannya dengan sempurna.

Ilustrasi: Meminta Komputer Membuat Teh Manis

  1. Ambil gelas.
  2. Ambil sendok.
  3. Masukkan kantung teh ke dalam gelas.
  4. Masukkan satu sendok teh gula ke dalam gelas.
  5. Nyalakan kompor.
  6. Taruh panci berisi air di atas kompor.
  7. Panaskan air sampai mendidih.
  8. Ambil air panas dari panci.
  9. Tuangkan air panas ke dalam gelas sampai penuh.
  10. Aduk teh dan gula menggunakan sendok.
  11. Angkat kantung teh dari gelas.
  12. Teh manis siap disajikan.

Semua instruksi ini harus diberikan agar komputer bisa menyelesaikan tugasnya tanpa kesalahan.

2. Apa Itu Code Editor? (VS Code)

Code Editor adalah alat yang kita gunakan untuk menulis kode. Anggaplah Code Editor seperti buku catatan digital khusus untuk programmer.

  • Bukan Buku Biasa: VS Code bukan buku catatan biasa seperti Notepad. Ia cerdas! Ia tahu aturan-aturan penulisan kode Python.
  • Pewarnaan Kode (Syntax Highlighting): VS Code akan mewarnai kode Anda. Misalnya, angka bisa berwarna biru, teks berwarna hijau, dan kata kunci penting berwarna ungu. Ini sangat membantu kita membaca dan menemukan kesalahan.
  • Terminal Langsung: Anda bisa menjalankan program langsung di dalam VS Code. Ini menghemat waktu karena Anda tidak perlu bolak-balik antara VS Code dan aplikasi lain.

Mengenal Komentar (#)

Komentar adalah teks yang kita tulis di dalam kode, tetapi tidak akan dibaca atau dijalankan oleh komputer. Komentar sangat penting bagi programmer.

  • Kegunaan Komentar:
    • Mengingatkan Diri Sendiri: Anda bisa menulis catatan untuk diri sendiri tentang apa tujuan dari sebuah baris kode atau bagian program. Ini akan membantu Anda mengingat kembali saat membuka kode tersebut di lain waktu.
    • Menjelaskan Kode untuk Orang Lain: Jika Anda bekerja dalam tim, komentar membantu teman-teman Anda memahami logika di balik kode yang Anda tulis.
    • Mematikan Kode Sementara: Jika Anda ingin mencoba sebuah ide baru dan tidak ingin menghapus kode lama, Anda bisa "mematikan" kode tersebut dengan mengubahnya menjadi komentar.

Cara Menulis Komentar:

  • Di Python, Anda bisa membuat komentar dengan menambahkan tanda pagar (#) di awal baris.
# Ini adalah komentar, tidak akan dijalankan oleh Python
print("Halo, dunia!") # Komentar juga bisa diletakkan di akhir baris

3. Apa Itu Bahasa Pemrograman & Python?

Bahasa pemrograman adalah bahasa yang kita gunakan untuk "berbicara" dengan komputer.

  • Kenapa ada banyak bahasa? Sama seperti manusia punya bahasa Indonesia, Inggris, atau Mandarin, komputer juga punya banyak bahasa. Setiap bahasa punya kelebihan masing-masing.
  • Python: Bahasa ini sangat terkenal karena sintaksnya yang sederhana. Kode Python sangat mirip dengan bahasa Inggris yang mudah dibaca. Ini membuat Python menjadi pilihan yang sangat baik untuk kita yang baru memulai.

4. Output di Python: Fungsi print()

Fungsi print() adalah cara kita meminta komputer untuk "berbicara" atau menampilkan sesuatu di layar. Ini adalah fungsi pertama yang harus kita kuasai.

Mencetak Teks:

Jika Anda ingin mencetak tulisan, tulisan tersebut harus diapit oleh tanda kutip ganda (" ") atau tunggal (' ').

print("Halo, dunia!")

Output:

Halo, dunia!

Mencetak Angka:

Jika Anda ingin mencetak angka atau hasil perhitungan, Anda tidak perlu menggunakan tanda kutip.

print(10 + 5)

Output:

15

Mencetak Banyak Hal Sekaligus:

Anda juga bisa menggabungkan teks dan angka dalam satu baris print() dengan menggunakan koma. Python akan secara otomatis menambahkan spasi di antara mereka.

print("Saya berusia", 13, "tahun.")

Output:

Saya berusia 13 tahun.

5. Operasi Aritmatika di Python

Python dapat berfungsi layaknya kalkulator canggih. Ia bisa melakukan berbagai perhitungan dasar. Mari kita lihat satu per satu.

  • Penjumlahan (+): Menggunakan simbol +
    • print(10 + 5)
  • Pengurangan (-): Menggunakan simbol -
    • print(15 - 8)
  • Perkalian (*): Menggunakan simbol *
    • print(4 * 6)
  • Pembagian (/): Menggunakan simbol /
    • print(20 / 4)
  • Pangkat (**): Menggunakan dua tanda bintang **
    • print(2 ** 5)

    Ini sama dengan 2 x 2 x 2 x 2 x 2.
    Output: 32

6. Variabel di Python

Variabel adalah seperti kotak penyimpanan yang bisa kita beri nama. Kita bisa menaruh data (seperti teks atau angka) di dalam kotak ini, lalu mengambilnya lagi kapan saja kita butuhkan.

Konsep Penting tentang Variabel

  1. Variabel Harus Didefinisikan Dulu:

    Komputer membaca kode dari baris paling atas ke paling bawah. Jadi, Anda harus membuat variabel (kotaknya) terlebih dahulu sebelum bisa menggunakannya.

    # Kode ini akan error, karena variabel 'nama' belum ada
    print(nama)
    nama = "Budi"

    Kode yang benar:

    nama = "Budi"
    print(nama)
  2. Isi Variabel Bisa Diubah:

    Ini adalah kekuatan utama dari variabel. Anda bisa mengubah isinya sesuai kebutuhan.

    # Di awal, isi kotak 'skor' adalah 100
    skor = 100
    print("Skor awal:", skor)

    # Sekarang, isi kotak 'skor' kita ubah menjadi 150
    skor = 150
    print("Skor baru:", skor)
  3. Aturan Penamaan Variabel:
    • Aturan Mutlak:
      • Nama variabel harus dimulai dengan huruf (a-z, A-Z) atau garis bawah (_).
      • Nama variabel tidak boleh dimulai dengan angka.
    • Praktik Terbaik (Best Practice):
      • Gunakan huruf kecil.
      • Jika nama variabel terdiri dari dua kata atau lebih, gunakan garis bawah (_) untuk memisahkan kata-kata tersebut. Ini disebut snake_case. Contoh: panjang_persegi.

7. String di Python

String adalah istilah untuk teks dalam pemrograman. Setiap kali Anda melihat tulisan yang diapit oleh tanda kutip, itu adalah string.

  • Mengapa harus ada tanda kutip?

    Tanda kutip memberitahu Python bahwa ini adalah teks biasa, bukan perintah atau variabel.

    Misalnya, jika Anda menulis print(nama), Python tahu nama adalah variabel. Tapi jika Anda menulis print("nama"), Python akan mencetak teks "nama".

Menggabungkan dan Mengulang String

  • Menggabungkan String (+):
    Anda bisa menggabungkan dua string menggunakan tanda plus (+).
    sapaan = "Selamat pagi,"
    nama = "Andi"
    print(sapaan + " " + nama)
    Output:
    Selamat pagi, Andi
  • Mengulang String (*):
    Anda bisa mengulang string menggunakan tanda kali (*).
    print("Wow" * 3)
    Output:
    WowWowWow
